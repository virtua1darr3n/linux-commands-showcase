# ls

## What it does
Lists files and directories.

## Scenario
You need to check what files exist in a directory and verify permissions and hidden files.

## Basic Usage
```bash
ls
```

## More detailed view
```bash
ls -la
```

## Notes
- `-l` = long format (permissions, owner, size)
- `-a` = include hidden files
